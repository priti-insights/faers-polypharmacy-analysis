# FAERS Polypharmacy Analysis

## Overview

This project investigates polypharmacy patterns in elderly patients using the FDA Adverse Event Reporting System (FAERS).

The workflow includes:

- Importing FAERS data (2022–2025)
- Data cleaning
- Latest case version selection
- Drug standardization using RxNorm
- Patient-level dataset generation
- Machine Learning for ADR and Serious Outcome prediction
- Temporal Stability Analysis

## Technologies

- PostgreSQL
- SQL
- Python
- Pandas
- Scikit-learn
- XGBoost
